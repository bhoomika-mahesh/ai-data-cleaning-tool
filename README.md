# 🧠 AI Data Cleaning Tool
🚀 Live App: https://huggingface.co/spaces/bhoomika10/data-cleaning-env

An interactive web application that simulates step-by-step data cleaning operations.

## 🚀 Features
- 📂 Upload CSV files
- 🧹 Remove missing values
- 🔁 Remove duplicates
- ⚙️ Normalize data (simulation)
- 📊 View data before and after cleaning
- 🔄 Reset and experiment with different steps

## 🎯 Objective
Instead of running scripts, this project provides a visual and interactive way to understand how data preprocessing works.

## 🛠️ Tech Stack
- Python
- Gradio
- Pandas (optional in advanced version)

## 🌐 Live Demo
[Hugging Face Space](PASTE_YOUR_LINK_HERE)

## 💡 Key Idea
Transform a basic data cleaning process into an interactive system that allows users to apply and visualize preprocessing steps dynamically.
